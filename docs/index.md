# QuinkGL Documentation

QuinkGL is a decentralized gossip learning framework for P2P edge intelligence.

## Decision Tree

- Are you new to QuinkGL? → [Getting started](getting-started.md) (or [Türkçe](getting-started-tr.md)), then [Tutorial T1](tutorials/T1/index.md)
- Are you a swarm creator? → [User Guide](user-guide/index.md) + Tutorial T2
- Are you a peer operator? → [User Guide](user-guide/index.md) + [CLI Reference](cli/index.md)
- Need a custom model/data? → Tutorial T4 + [API Reference](reference/index.md)
- Building in production? → Security Model + Migration Guide
- Debugging an error? → [User Guide Troubleshooting](user-guide/troubleshooting.md) — look up `ERR_*`
